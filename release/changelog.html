Welcome to the WebExtensions re-write of Neo Diggler.

If you have any problems, check the bug-list on GitHub first, but do feel free to email me (or create a new bug report).
Please don't use Mozilla Add-ons site reviews as a bug-reporting tool.

See the add-on page for an explanation of required permissions.

PLEASE NOTE that there have had to be some concessions to the way Neo Diggler works due to limitations imposed by the WebExtensions architecture. I can do nothing about these, as yet, it is inherent in Firefox. I hope the alternatives I have provided work for you; if you have any ideas for improvements, please let me know!

The main changes from the old version are:

  • WebExtensions cannot affect standard browser options, so the image, JavaScript and Popup permission toggles had to go.
  • WebExtensions cannot affect the URL bar, so I cannot directly clear it. Instead, I provide a blank popup and a “go” button to try to recreate the old behaviour. However for security reasons the URLs that this can access are limited (to ftp/http/https). If the URL you enter is prevented by Firefox, it will flash red.
  • The old menu is now a context menu; the top-level context menus for browser buttons has a maximum of 6 extra items, so I have had to push the menu list to a sub-menu which has no such limitation. But! I have added the ability to have a page-action button instead of a normal one (inside the URL bar, on the right). You can turn this off in preferences, or, if you prefer it, drag the old-style button off the display while in customise mode.
